SJSU Spartan Food Pantry

Real-Time Webpage

Overview

I volunteered at the Spartan Food Pantry for a year, which inspired me to create a website for my senior project. My experience at the pantry highlighted the need for a more organized and efficient shopping process, motivating me to simplify it.


Responsibilities

Project Context

  • Fullstack Developer
  • February 2023 - August 2023
  • Team project

The Problem

The Spartan Food Pantry offers food assistance to San Jose State University students, providing non-perishable items, fresh produce, and refrigerated goods. Students frequently visit the pantry seeking specific items, such as milk and eggs.

Because the pantry's inventory varies, creating a real-time web page displaying current offerings would enable students to check what's available before visiting.

Ideation and sketch

    1. Brainstormed a lesson topic and devised a plan to introduce a computer science concept to students.

    2. Constructed engaging PowerPoint presentations with clear agendas and goals, introducing subjects effectively, and using interactive class activities to explore concepts.

    3. Developed an engaging game using Scratch that captivates and motivates the youth to actively participate in coding and complete the necessary steps for the game to function.

    4. Created step-by-step directions for students to follow, enabling them to learn and construct the code.

The Solution

Through this project, I learned the importance of teamwork in achieving project success. Moreover, it served as a practical learning experience that not only familiarized me with game design, but also equipped me with the essential skills to implement code using Scratch.
